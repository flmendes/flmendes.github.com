Efetue o Download do XDebug através do site http://xdebug.org/

Não esqueça de verificar a versão do seu PHP, no meu caso estou utilizando o pacote mais recente do wampserver que já vem com a versão 5.3.

Agora copie a dll xdebug que você baixou para a pasta ext do php, segue um exemplo "C:/wamp/bin/php/php5.3.0/ext/"

Pronto agora você pode adicionar a configura abaixo em seu arquivo php.ini.

[XDebug]

zend_extension="C:/wamp/bin/php/php5.3.0/ext/php_xdebug-2.0.5-5.3-vc6.dll"
xdebug.remote_enable=true
xdebug.remote_host=127.0.0.1  ; if debugging on remote server,
                              ; put client IP here
xdebug.remote_port=9000
xdebug.remote_handler=dbgp
xdebug.profiler_enable=0
xdebug.profiler_output_dir="C:/wamp/"
Resete o wampserver!
Cuidado!
Se a sua versão do php não for a 5.3 então o parametro zend_extension deve ser alterado para zend_extension_ts.
Se o servidor está funcionando corretamente então basta verificar no phpinfo() se existe uma entrada do xdebug, geralmente fica na tela de powered by do Zend Engine 2.
