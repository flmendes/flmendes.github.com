Hoje precisei de consumir um web service em SOAP utilizando Python.

Para esta tarefa encontrei uma pequena lib chamada SUDS(https://fedorahosted.org/suds/). O SUDS é um cliente leve e simples escrito em Python

Segue um pequeno cliente.

from suds import WebFault
from suds.client import Clientfrom suds.transport.http import HttpAuthenticatedfrom suds.xsd.doctor import ImportDoctor, Import

print client.service.LatLonListZipCode('90210')

Sem o uso do doctor o cliente as vezes recebe um erro de encodding.
A classe HttpAuthenticated serve para auxiliar na autenticação do usuário junto ao serviço agora um cuidado extra, não estamos utilizando conexão segura então o pacote desta classe é suds.transport.http.
Se a conexão for segura o pacote muda para suds.transport.https e o nome da classe permanece o mesmo.
