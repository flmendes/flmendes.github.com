Uma pasta que sempre me enche o saco é o ".DS_Store", eu sei que para remover essa pasta basta adicionar as seguinte linhas em seu arquivo .gitignore.

.DS_Store

Parece bem simples, mas é algo que sempre esqueço de fazer antes do primeiro commit do projeto.

Se você esqueceu de adicionar o diretório em seu .gitignore e realizou um commit/push, segue uma dica para remover esta pasta.

Execute o seguinte comando: find . -name *.DS_Store -type f -exec git rm  -rf {} \;

Esse comando só funciona no Linux/Mac e afins. O comando irá pesquisar nos diretórios do seu projeto e remover o .DS_Store do controle de versão.

Agora basta adicionar as linhas no .gitignore e efetuar um commit/push, o mesmo se aplica para o svn.
