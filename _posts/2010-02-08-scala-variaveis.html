Continuando nos primeiros passos da linguagem scala, hoje vamos dar uma olhada em suas variáveis.
Todas as variáveis no scala são objetos incluindo os valores.
Os principais tipos são:

  • scala.Double
  • scala.Float
  • scala.Long
  • scala.Int
  • scala.Short
  • scala.Byte
  • scala.Boolean
  • scala.Unit
  • scala.Char
  • scala.String
  • scala.Null
  • scala.Noting

para criar uma variável temos duas palavras chave, val e var. A palavra chave Val cria uma variável imutável, já a var cria uma variável mutável.

scala> val nome="flavio luiz mendes"
nome: java.lang.String = flavio luiz Mendes

scala> val sexo = 'M'
sexo: Char = M

scala> nome = "Flavio"
<console>:5: error: reassignment to val
       nome = "Flavio"

scala> var salario = 1500.50
salario: Double = 1500.5

scala> salario=150000.00
salario: Double = 150000.0

Reparou que a variável do tipo String veio do java.lang.String, no scala conseguimos utilizar todos os objetos do Java.
