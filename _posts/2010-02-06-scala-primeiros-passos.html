Introdução

Scala é uma linguagem de programação de uso geral projetado para expressar padrões comuns de programação em uma forma concisa e elegante. A linguagem de programação Scala utiliza a plataforma Java sendo assim ela utiliza todos recursos do Java, o compilador da Scala gera byte code Java então podemos utilizar todos os recursos disponíveis pela API Java.

A linguagem de programação Scala é orientada a objeto,  funcional, extensível,  fortemente tipada com tipagem estática e focada em expressividade.

Instalação

A instalação da linguagem é bastante simples, basta efetuar o download do pacote conforme o seu sistema operacional no link abaixo.

http://www.scala-lang.org/downloads

Para instalar basta descompactar o arquivo e criar uma variável de ambiente apontando para o diretório gerado ao descompactar o arquivo.

Exemplo:

/Users/flavio/Development/scala-2.7.7.final

O diretório acima será o valor da sua variável de ambiente “SCALA_HOME”.

Após criar está variável modifique o PATH do sistema para que fique da seguinte forma.

PATH:$PATH:$SCALA_HOME/bin

Com a adição acima termos a disposição o compilador da linguagem e alguns utilitários.
