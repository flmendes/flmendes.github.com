Salve galera, depois de anos sonhando com um Mac Book Pro, a uns meses atrás eu consegui realizar esse sonho.

Mas quando comecei a trabalhar com o notebook, o sonho virou um pesadelo. O miserável do eclipse travava toda hora.

Depois de um tempo pesquisando, achei uma dica interessante que deixou o eclipse mais rápido.

No diretório raiz do eclipse existe um arquivo chamado eclipse.ini, este arquivo é responsável pelas configurações básicas, do eclipse.

As configurações de “fábrica” são modestas, então é necessário alterar alguns detalhes.

O eclipse instalado em minha máquina é a versão Helios, a pasta eclipse foi movida para o diretório Applications, então o path do arquivo é o seguinte:

/Applications/eclipse/Eclipse.app/Contents/MacOS/eclipse.ini

Edite o arquivo acima e altere as seguintes propriedades:

-XX:MaxPermSize=512m
-Xms128m
-Xmx1024m
-Dosgi.requiredJavaVersion=1.6

O Xms é o tamanho inicial da memoria Heap do Java e o Xmx é o tamanho máximo. O -Dosgi.requiredJavaVersion é a versão da JVM utilizada.

Essa configuração serve para qualquer sistema operacional, a única diferença é o caminho do arquivo eclipse.ini. Essa configuração também pode ser feita no Zend Studio, Aptana, Eclipse PDT etc.
